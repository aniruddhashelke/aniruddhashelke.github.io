NYC cabs data engineering

This is one my favorite projects that I have worked on. I have tried to use different technologies in all my projects to get acquainted to various technologies. This time was no different. I used the cloud serivces offered by Google, The Goolge Cloud Platform for this project. The main ETL pipleline was built using MAGE AI, which is an open source, AI enabled data orchestration tool to easily build pipelines. It writed all the code to build the pipeline for you based on the parameters that you give.

The dataset was a flat csv file with all the information in one table. The first task was to normalize the table into different fact and dimension tables and the defining the realtions between the tables in a STAR Schema. The data transformation was done using Python on MAGE AI.

After the dataset was tranformed. I uploaded the dataset on Google BigQuery, which is the data warehouse service offered by GCP. I wrote SQL queries to analyze the data which would help me build the dashboard. Then I built a dashboard using Google Looker Studio.

The dataset contained information about the green and yellow cabs in New York City. Nyc cabs have different rates based on different regions. It was interesting to find out average trip distance, fare amounts and tip amounts based on different payment methods.
