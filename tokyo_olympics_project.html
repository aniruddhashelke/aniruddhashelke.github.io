Tokyo Olympics Data engineering

The goal of this project was to build a pipeline using the resources provided by Microsoft's Azure cloud. Once you get to know the services offered by one cloud, migrating to another isn't very difficult. Most clouds offer similar services with slight nuances and changes in UI. This is still an ongoing project and I haven't built a dashboard for this project yet, but I am planning to build one very soon. The tool I am planning to use is Power BI, continuing with the theme of using services offered by Microsoft.

The data was taken from Kaggle. There were 5 csv files contaning information of the players, teams, coaches, information about no. of medals won by each country and information about the gender distribution of the participants. I made my free Azure account and started with this project. I ingested the data into Azure Data Lake Storage. I then used Azure Data Factory as the data orchestration took for this project. I used an API to get the data which was uploaded on my GitHub page.

Then it was time for data transformation. I used Azure Databricks which is a collaboration between Azure and Databricks to provide Data tranformation services offered by Databricks on the azure cloud itself. I did very basic data trasformation using PySpark to change the schema of the data which will help in quantitative analysis in the future.
